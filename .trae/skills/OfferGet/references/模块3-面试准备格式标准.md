# 模块3：面试准备（精简版）格式标准

## 输入标准

**用户提供：**
- JD文本
- 公司名称
- 岗位名称

**AI读取：**
- 从 `for offer/02-准备/04-经历深挖/` 读取对应经历的STAR完整版
- 从 `for offer/02-准备/04-经历深挖/` 读取追问预案
- 从 `for offer/02-准备/02-自我介绍.md` 读取自我介绍（作为参考，不重复生成）

## 设计原则

1. **避免重复**：不生成宝洁八大问、通用自我介绍等已在其他文件里的内容
2. **基于JD预测**：只生成针对该JD的高频问题
3. **引用而非复制**：引用04-经历深挖的内容，而不是重新生成
4. **针对性**：所有内容都针对特定公司和岗位

## 输出内容

**不包含（避免重复）：**
- ❌ 宝洁八大问（已在 `02-自我介绍.md`）
- ❌ 通用自我介绍（已在 `02-自我介绍.md`）
- ❌ 通用优缺点（已在 `02-自我介绍.md`）

**包含（针对性内容）：**
- ✅ 针对该JD的自我介绍版本（1分钟，可直接背诵）
- ✅ 基于JD的高频问题（4-6个，不是通用问题）
- ✅ 基于经历的追问预案（引用04-经历深挖）
- ✅ 针对该公司的三个Why
- ✅ 参考文件链接（指向已有素材）

1. **不重复**：不生成已在其他文件里的通用内容

3. **可引用**：追问预案引用04-经历深挖的内容，不重新生成
4. **可直接使用**：答案要可以直接背诵
5. **结构清晰**：包含参考文件链接，方便用户查找更多内容
