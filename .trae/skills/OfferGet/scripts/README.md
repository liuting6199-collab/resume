# 自动化脚本说明

## 脚本列表

### 1. create-folder-structure.ps1
自动创建求职准备文件夹结构

### 2. generate-html-from-template.ps1
基于模板生成HTML文件

## 使用方法

### 创建文件夹结构
```powershell
.\scripts\create-folder-structure.ps1
```

### 生成HTML文件
```powershell
.\scripts\generate-html-from-template.ps1 -CompanyName "字节跳动" -Position "AI数据策略专家"
```
