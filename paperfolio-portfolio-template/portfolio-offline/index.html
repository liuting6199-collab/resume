Portfolio Project 01

电商用户转化漏斗分析与流失定位

独立分析

基于1,500条用户行为数据，拆解全链路转化，定位最大流失环节并提出优化策略

PythonPandas漏斗分析渠道归因用户分层Matplotlib

项目背景与目标

某电商平台1月份产生1,500条用户行为记录，覆盖浏览、点击、加购、下单、支付全链路。发现整体转化率低于行业均值，希望定位流失最严重的环节，针对性优化。

核心问题：

  • 哪个环节流失用户最多？流失率是多少？
  • 不同渠道的转化差异？
  • 不同品类、设备、用户类型的转化表现？
  • 如何量化优化后的潜在收益？

数据说明与清洗

数据集包含1,500条用户行为记录，覆盖2024年1月1日-1月31日。每条记录包含用户ID、设备类型、来源渠道、商品品类、用户类型、行为链路及订单金额。

数据字段

user_id · device · channel · category · user_type · view/click/cart/order/pay · amount · date

清洗过程

① 去重：无重复记录
② 缺失值：无缺失字段
③ 异常值：订单金额在合理范围内
④ 逻辑校验：行为链路符合顺序

分析思路与假设

从原始行为日志出发，按「漏斗概览 → 流失定位 → 渠道拆解 → 品类对比 → 用户分层 → 设备分析」逐层下钻。

漏斗概览→流失定位→渠道拆解→品类对比→用户分层→设备分析

核心假设：① 百亿补贴渠道「高点击、低加购」；② 活动页渠道转化效率最低；③ 会员用户转化率显著高于新用户；④ 移动端转化率低于PC端。

数据可视化与分析

核心发现

  • →点击→加购是最大流失环节（56.9%）：1,028个点击用户中仅443个加购，超过一半用户点击后流失。说明商品详情页吸引力不足。
  • →分享渠道「少而精」，活动页完全无效：分享渠道整体转化率23.6%为全渠道最高，而活动页仅3.2%。建议暂停活动页发券。
  • →百亿补贴「高点击、低加购」：点击率85%全渠道最高，但加购率仅35%，用户被低价吸引但决策犹豫。
  • →会员转化率16.2% vs 新用户11.8%：会员用户转化率比新用户高37%，建议加大会员权益投入。

业务建议与预期收益

优化商品详情页（ROI最高）

针对点击→加购流失56.9%，优化详情页首屏信息、增加用户评价。若加购率提升至55%，预计每月增收¥20,000+。

暂停活动页发券

活动页转化率仅3.2%，将预算转移至分享和搜索渠道。预计整体转化率提升至16.5%。

百亿补贴加紧迫感

增加「仅剩XX件」等提示，利用损失厌恶心理。若加购率从35%提升至45%，预计每月增加订单120+。

扩大分享裂变机制

分享渠道转化效率最高（23.6%），建议增加分享奖励。预计分享流量占比可从14%提升至20%。

会员权益升级

会员转化率比新用户高37%，建议推出会员专属价、免邮券等权益。

移动端体验优化

PC端转化率（18.5%）比移动端（13.2%）高40%，需优化移动端体验。

分析局限性与后续方向

  • 数据周期：仅覆盖1个月数据，建议补充季度数据验证结论稳定性。
  • 用户画像：缺少年龄、地域等维度，无法做更精细的用户分层。
  • 因果推断：当前为相关性分析，建议通过A/B测试验证优化效果。
  • 竞品对比：缺少行业基准数据，无法判断转化率相对位置。

数据来源：ecommerce_data_1500.csv | 分析脚本：generate_ecommerce_data.py

Portfolio Project 02

优惠券投放ROI优化与用户分层分析

独立分析

基于1,200条优惠券发放记录，评估不同券类型、渠道、用户群体的ROI表现，提出精准投放策略

PythonPandasRFM模型ROI分析用户分层A/B测试设计

项目背景与目标

某外卖/到店平台每月发放大量优惠券，但缺乏系统的ROI评估体系。业务方希望了解：哪种券类型ROI最高？哪些用户群体对优惠券最敏感？

核心问题：

  • 满减券/折扣券/无门槛券，哪种ROI最高？
  • APP推送/短信/微信，哪个渠道发券最有效？
  • 新用户/老用户/会员，谁对优惠券更敏感？
  • 如何设计分层投放策略，实现补贴效率最大化？

数据说明与清洗

数据集包含1,200条优惠券发放记录，覆盖2024年2月。每条记录包含用户ID、券类型、发放渠道、用户类型、是否核销、订单金额、补贴金额等字段。

数据字段

user_id · coupon_type · channel · user_type · is_used · order_amount · subsidy_amount · roi

清洗过程

① 去重：无重复发放记录
② 缺失值：无缺失字段
③ 异常值：ROI在0-20范围内
④ 逻辑校验：核销订单必有金额

分析思路与假设

从原始发放记录出发，按「券类型对比 → 渠道效果 → 用户分层 → 交叉分析 → 策略设计」逐层深入。

券类型对比→渠道效果→用户分层→交叉分析→策略设计

核心假设：① 折扣券ROI最高但核销率低；② APP推送和短信渠道核销率最高；③ 会员用户对优惠券敏感度低于新用户；④ 满减券适合拉新，折扣券适合促活。

数据可视化与分析

核心发现

  • →折扣券ROI最高（5.18）但核销率最低（54.5%）：折扣券吸引高价值用户，每补贴1元带来5.18元订单，但覆盖范围有限。适合精准投放。
  • →满减券核销率最高（75.0%）但ROI中等（3.92）：满减券门槛低，用户易达标，适合拉新和促活。建议作为新用户首单券。
  • →APP推送和短信渠道核销率均为100%：精准推送渠道效果远超泛流量渠道。建议暂停活动页发券。
  • →会员核销率100% vs 普通用户35.7%：会员用户对平台信任度高，使用意愿强。建议扩大会员规模。

分层投放策略与预期收益

新用户：满减券 + APP推送

发放「满30减10」低门槛券，通过APP推送精准触达。预计新用户转化率从11.8%提升至25%。

老用户：折扣券 + 短信提醒

发放「8折券」（最高减30），通过短信+APP双渠道提醒。预计老用户复购率提升15%。

会员：专属折扣券 + 生日礼

发放「7折会员专属券」+生日月双倍积分。预计会员客单价提升20%。

沉睡用户：无门槛券 + 微信触达

发放「5元无门槛券」通过微信唤醒。预计沉睡用户召回率10%。

暂停活动页发券

活动页核销率0%，立即暂停。将预算转移至APP推送和短信渠道。

建立券效监控看板

实时监控各券类型、渠道、用户群的ROI和核销率，每周迭代优化投放策略。

A/B测试验证方案

测试1：券类型效果验证

假设：折扣券ROI > 满减券ROI
实验组：500人发放折扣券
对照组：500人发放满减券
指标：ROI、核销率、客单价
周期：2周 | 成功标准：ROI提升≥15%

测试2：渠道效果验证

假设：APP推送核销率 > 短信核销率
实验组：300人APP推送发券
对照组：300人短信发券
指标：核销率、7日留存、30日留存
周期：1周 | 成功标准：核销率提升≥10%

分析局限性与后续方向

  • 数据周期：仅覆盖1个月数据，无法观察优惠券的长期效果。
  • 券类型有限：缺少新人券、裂变券等新型优惠券的数据。
  • 因果推断：当前为相关性分析，无法完全排除选择偏差。
  • 外部因素：未考虑竞品活动、节假日等外部因素的影响。
